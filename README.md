# Sanjay Portfolio

Personal portfolio website for Kodimyala Sanjay, built with React and Vite.

Live site:
https://sanjaykodimyala.github.io/sanjayportfolio/

GitHub repository:
https://github.com/sanjaykodimyala/sanjayportfolio

## Local Project Folder

The project is stored on this computer at:

```text
C:\Users\kodim\sanjayportfolio
```

## Useful Commands

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
